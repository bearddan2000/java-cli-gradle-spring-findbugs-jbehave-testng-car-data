# java-cli-gradle-spring-findbugs-jbehave-testng-car-data

## Description
A POC for a spring app using testng,
findbugs, jbehave framework.

## Tech stack
- java
- gradle
	- spring
  - testng
  - findbugs
  - jbehave

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
